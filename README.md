# Housing Price Prediction with LSTM

## Overview
This notebook demonstrates a step-by-step process for predicting housing prices using time series data. It merges housing price indices with CPI data, applies data cleaning and normalization, and trains an LSTM neural network to forecast future housing prices across different property types.

## Features
- Data merging and preprocessing of housing price and CPI datasets
- Exploratory data analysis with visualizations
- Data transformation for time series modeling
- LSTM model construction and training for multi-type house price prediction
- Evaluation with R² score, Mean Squared Error, and Root Mean Squared Error
- Visualization of actual vs predicted housing prices

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- Libraries: pandas, numpy, scikit-learn, tensorflow (Keras), matplotlib, seaborn

## How to Use
1. Clone the repository.
2. Upload the housing price and CPI datasets to your environment or update paths accordingly.
3. Open the notebook `housing-price-prediction-lstm.ipynb` in Jupyter or Google Colab.
4. Run all cells to preprocess data, train the LSTM model, and evaluate predictions.
5. Modify sequence length or model parameters to experiment with results.
